# BitBasis

A modern Bitcoin portfolio tracking and cost basis analysis application built with Next.js.

## Features

- Dashboard with portfolio overview and performance metrics
- Cost basis tracking and analysis
- Transaction history management
- Performance analytics and charts
- Responsive design for desktop and mobile
- Dark mode support

## Tech Stack

- Next.js 14
- React
- TypeScript
- Tailwind CSS
- Shadcn/ui Components

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/brettwhite-git/bitbasis.git
```

2. Install dependencies:
```bash
cd bitbasis
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Development

The project uses:
- TypeScript for type safety
- ESLint for code linting
- Prettier for code formatting
- Tailwind CSS for styling
